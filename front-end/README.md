# 前端安装与启动

### 环境配置
- 安装node.js和vue-cli
- 使用npm安装yarn
  ```bash
  npm install -g yarn
  ```
- **注意：** 由于模板版本较老，相关的包依赖于`python2`和`visual studio 2017`。其中`python2`推荐用`anaconda`创建新环境，`visual studio 2017`可以通过`visual studio installer`安装，多个VS版本是可以**共存**的。
- 使用yarn安装依赖包
  ```bash
  yarn
  ```
### 启动
执行下列命令。
```bash
yarn serve
```
